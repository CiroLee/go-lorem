# Vrm      
return a random Chinese vehicle registration mark       
signature:        
```go
func Vrm() string
```
example:      
```go
lorem.Vrm() // 陕O7LPQ港
```

## Vin          
return a random vehicle identification number         
signature:     
```go
func Vin() string
```
example:      
```go
lorem.Vin()  // 2F4DG84G3RP840296
```

## Id        
return a random Chinese Mainland ID          
signature:        
```go
func Id() string
```
example:      
```go
lorem.Id() // 985894197001014122
```
